# Project Roadmap

This roadmap outlines planned features and milestones for BlackboxProgramming. Dates are tentative and subject to change as development progresses.

## Milestones

- **MVP Release (Q4 2025)** – Deliver a basic collaborative editor with user authentication and an AI code assistant integrated.
- **Real-time Collaboration Enhancements (Q1 2026)** – Improve low-latency editing, add conflict resolution UI, and presence indicators.
- **AI Agent Expansion (Q2 2026)** – Introduce new agents for code review, automated testing, and documentation generation.
- **Documentation & Docs Site (Q3 2026)** – Publish comprehensive documentation via MkDocs and migrate design decisions into ADRs.

Feel free to propose new features or milestones by opening an issue or discussion.
